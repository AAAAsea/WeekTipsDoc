# WeekTips详解
## 什么是WeekTips
::: tip
WeekTips是一款基于Uniapp的由个人开发的一款非常简洁好用的**周计划待办软件**
:::
## 项目地址
[GitHub地址](https://github.com/AAAAsea/weektips/releases)
## 安装使用
[安卓版本下载地址](https://github.com/AAAAsea/weektips/releases/download/1.0.4.8/WeekTips_1.0.4.8.apk)（暂不支持ios用户）
